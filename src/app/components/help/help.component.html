Troubleshooting
Profile issues?

What do you if you can't see your profile on the profile page? Refresh the page and wait a few seconds

User chat problems?

Did your text fail to send or you deleted the wrong message?, try to refresh the page. If the page fails to load try reloading the website in a new tab.

Friend list trouble?

Did you accidentally delete someone off your friends list?, if that happens you can use the Restore list button underneath the friends list.

Note problems?

Did your Note to a friend fail to connect with the chat system?,Don't worry.Try refreshing the page or use the Resend Note button.
